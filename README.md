# Cafe minu

> 카페는 가고 싶은데 북적이는 곳으로의 외출이 힘든 내향인이 만든 홈 카페

🔗 https://daehoidar.github.io

---

### ☕ 메뉴
마시고 싶은 음료를 주문할 수 있습니다.

### 🖼 갤러리
주인장이 직접 찍은 사진이 전시됩니다.

### 📖 방명록
이름과 메시지를 남길 수 있습니다.

> 방명록은 이제 `guestbook-backend`(FastAPI) API를 통해 동작합니다.
> 로컬 실행/배포는 `guestbook-backend/README.md`를 참고하세요.

### 🎵 배경 음악
주인장의 지극히 개인적인 취향이 담긴 플레이리스트.

### 🐱 이스터에그
- 고양이 클릭 → 모자 선물 / 노래 스킵
